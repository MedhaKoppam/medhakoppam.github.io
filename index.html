Bio

I am a Research Fellow at Microsoft Research India, where I work on agentic and deterministic systems for large-scale code intelligence, focusing on automated workflow generation, unit test generation, and code transformations. My work spans LLM orchestration, context retrieval using code graphs, vector indexing, and building tools that support scalable software engineering.

Previously, I interned at Couchbase and Rakuten, where I explored how AI techniques can augment systems engineering — from prompt decomposition using knowledge-graph-driven RAG pipelines to AI-assisted SQL and code optimization. I also published work on educational technology and multilingual summarization through my capstone project Edvent.

My research interests lie in the intersection of AI, developer tooling, distributed systems, and automation for large-scale software development. I aim to pursue a Ph.D. in the future after gaining deeper experience in research-driven engineering roles.
